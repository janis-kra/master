\section{Anmeldung}
\label{task0}

Öffnen Sie einen Internet Browser Ihrer Wahl mit dem Sie vertraut sind -- etwa Google Chrome, Firefox, oder Safari -- und navigieren sie zu \url{http://localhost}.
Nach Laden der Seite sehen Sie einen Anmeldebildschirm.

\textbf{Aufgabe:} Geben Sie die folgenden Benutzerdaten ein und klicken Sie anschlie\ss end auf den mit ,,Anmelden'' betitelten Knopf:

\begin{itemize}
\item Benutzername: karl
\item Passwort: 12345678
\end{itemize}

\textbf{Wichtig:} Prüfen Sie bitte, ob Sie auf der linken Seite des Fenster ein blaues Menü sehen können.
Falls nicht, vergrö\ss ern Sie Ihr Browserfenster bis das Menü auf der linken Seite erscheint (s. Bildschirmfoto \ref{menu-example}).

\section{Einführung}
\label{task1}

Falls Sie noch nie eine webbasierte Chatanwendung verwendet haben oder einfach nur eine kurze Einführung in Mattermost erhalten möchten, können Sie sich jetzt entscheiden, ob Sie eine kurze Einführung durchlaufen möchten.

\textbf{Aufgabe}: Falls Sie die Einführung durchlaufen möchten, können Sie jetzt den Einführungstext von Mattermost lesen und danach auf den ,,Weiter'' betitelten Knopf anklicken.
Wenn Sie die Einführung überspringen möchten, klicken Sie einfach auf ,,Anleitung überspringen''.

Wenn Sie damit fertig sind und ein Chatfenster mit Katzenbildern sehen können (weil Sie initial im "Katzen" Kanal landen), können sie mit Aufgabe \ref{task2} weitermachen.
Ab jetzt dürfen Sie auch gerne nach Belieben den Chatverlauf ansehen oder Nachrichten schreiben.

\section{Kanal wechseln}
\label{task2}

Im Menü auf der linken Seite des Fenster können Sie eine Liste öffentlicher Kanäle sehen.
Ein Kanal ist eine Chatgruppe der Sie beitreten können um dort Nachrichten lesen und schreiben zu können.
Es gibt im Moment nur zwei Kanäle: ,,Katzen'' und ,,Hunde''.
Sie sind aktuell im Katzen Kanal.

\textbf{Aufgabe}: Klicken Sie im linksseitigen Menü den Eintrag für den Hunde Kanal an, um den Kanal zu wechseln.
Wenn sich die Inhalte des Hauptfensters verändern und dort Hundebilder zu sehen sind, war der Kanalwechsel erfolgreich.

\subsection{Verwendung des Kanalwechslers}

Man kann auch den ,,Kanalwechsler'' verwenden, um zu einem anderen Kanal zu wechseln.
Sie können den Kanalwechsler aufrufen, in dem sie entweder auf ,,Kanal wechseln'' in der unteren linken Ecke des Fenster klicken, oder Strg+K bzw. Cmd+K drücken (je nachdem ob Sie einen Windows PC oder Mac verwenden).
Im Kanalwechsler können Sie den Namen des gewünschten Kanals eingeben (oder nur einen Teil des Namens) und dann auf den Kanal klicken in den Sie wechseln wollen.

\textbf{Aufgabe}: Rufen Sie den Kanalwechsler auf in dem Sie auf ,,Kanal wechseln'' klicken; geben Sie dann ,,Katzen'' ein und klicken sie auf den erscheinenden Kanal.
Wenn Sie wieder die eingangs gezeigten Katzenbilder sehen, war der Kanalwechsel erfolgreich.

\subsection{Ein letztes mal den Kanal wechseln}

Da Hunde ganz offensichtlich besser sind als Katzen, wollen wir jetzt zurück zum Hunde Kanal wechseln.

\textbf{Aufgaben}: Bitte wechseln Sie also jetzt wieder zum Hunde Kanal, Sie können hierbei frei wählen über welche der erlenten Techniken das  passieren soll: Entweder über den Kanalwechsler unten links, oder über das linksseitige Menü.

Wenn Sie das getan haben, machen Sie bitte mit der finalen Aufgabe \ref{task4} weiter.

\section{Eine Nachricht senden}
\label{task4}

Natürlich ermöglich Mattermost auch das Verfassen und Versenden von Nachrichten in einem beliebigen Kanal.
Das können Sie tun, indem Sie in die Textbox am unteren Rand des Fensters klicken, Ihre Nachricht eintippen und anschlie\ss end die Enter-Taste drücken.

\textbf{Aufabe}: Wählen Sie zunächst einen der beiden kennengelernten Kanäle aus.
Schreiben Sie dann in diesem Kanal eine beliebige Nachricht und drücken Sie die Enter-Taste um die Nachricht zu versenden.
Der Inhalt der Nachricht ist nicht wichtig, ein einfaches ,,Hallo'' ist ausreichend.
